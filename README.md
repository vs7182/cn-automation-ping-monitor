# cn-automation-ping-monitor



This is a beginner-friendly project created to improve my skills in computer networks and Python automation.

The project monitors the availability of multiple IP addresses provided in a host file. It checks whether each device is reachable by sending a ping request and determines if the device is up or down.

The script also logs the results, including response time, and generates a summary report for better visibility.

The tool is also classified how fast over ip response based on response time
This tool also export the result as csv file for easy reporting and analyzing
